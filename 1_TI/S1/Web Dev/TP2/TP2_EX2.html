Chapitre 2: Mise en forme avec CSS

Presentation du CSS

Le role de CSS est de gerer l'apparence d la page web à travers la definition du style.

Definition des styles

Il y a 3 façons de definir des styles CSS:
-Directement dans les balises du fichier html : inline
-Dans l'entete de la page : interne
-Dans une feuille de style: externe
