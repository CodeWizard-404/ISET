Syntaxe de definition des styles

Definition des styles dans les balises

On ajoute un attribut styles à n'importe quelle balise.


Syntaxe:

< balise style ="propriete1: valeur; propriete2: valeur;..."></balise >

Definition des styles dans l'entête de la page

Cela consiste à inserer le code CSS directement dans une balise < style > à l'interieur de l'entête < head >
Syntaxe:

< head >

< style > selecteur {propriete1: valeur; propriete2: valeur;...}

< style >

< head >

Definition des styles dans une feuille de style

Les styles sont definis dans un fichier ayant pour extension .css
Syntaxe:

Il faut ajouter dans l'entête

&ly link rel="stylesheet" href="chemin vers le fichier.css"/ >
